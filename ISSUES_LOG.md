# User Feedback & Issues Log

1. **Feature Availability in Simple View**:
   - In the simple view (non-pro), the feature to add a reminder should be available.

2. **UI Refresh Issue**:
   - On updating a task (after changing anything and clicking update), when going back to the detail page, the detail page is not refreshing to show the new data.

3. **Past Task Completion**:
   - If a task is created in the past (e.g., on March 2), there might be a need to mark some of the past days as completed. Currently, there is no provision for that. Need to think of a good design to handle this.

4. **Bottom Nav Bar State**:
   - On opening any date, the app navigates to the task list for that date. However, the bottom navigation bar always highlights "Today". It should only highlight "Today" if the selected date is actually "Today".
